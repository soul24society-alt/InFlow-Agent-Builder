## 🧩 Usage Guide

### Backgrounds
- `#0D0D0D` — root page background
- `#111100` — nav, cards, surface elements
- `#1E1A00` — hover states, elevated cards, section backgrounds

### Typography
- `#F5C200` — all headings, brand name, section titles
- `#FFFFFF` — body copy, descriptions, subtitles
- `#6B5E00` — muted/secondary text, labels, metadata

### Interactive
- **Primary CTA**: `#F5C200` background + `#0D0D0D` text
- **Outlined CTA**: `#F5C200` border + `#F5C200` text on transparent
- **Danger/1st place**: `#E84040` background + `#FFFFFF` text
- **Info/2nd place**: `#1E6EF5` background + `#FFFFFF` text

### Borders & Dividers
- `#2A2200` — default border (0.5px–1px)

---

## Notes

- All backgrounds use **yellow-tinted blacks** (not pure black) — keeps everything warm and in-family with the gold
- `#F5C200` is the single brand color — use it for everything that needs attention
- Red (`#E84040`) and Blue (`#1E6EF5`) are **accent-only** — used for prize tiers and status badges, not UI chrome
- The palette has a deliberate **brutalist/retro-futuristic** feel — avoid rounded corners and soft shadows
